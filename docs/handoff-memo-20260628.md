- 웹슬롯에서 hovercard 같은 내부 subframe 이동 시 슬롯이 깨지는 문제 →
  `saveCurrentUrl`에 `event.isMainFrame === false` 가드 추가하면 해결됨.
  코드 검증 + 실사용 확인 모두 완료된 해법

- 웹슬롯 작업을 하고 나면 앱 재시작 시 기존 워크스테이션이 사이드바에서
  사라지고 Claude가 로그아웃 상태로 보이는 증상이 **1차, 2차 모두 재발**

  `workspaceStore.ts`의 strict guard) 모두 코드로 직접 고치고 검증까지
  했는데도 또 재발함 — 즉 **진짜 원인이 아직 따로 있음**
- `groupStore.ts`는 사실 `main.tsx`에서 호출 안 되는 죽은 코드였다는 것도
  이번에 확인됨 — 다음에 또 이 파일을 의심하기 전에 호출 여부부터 grep으로
  재확인할 것
